Daniel Agapov's Robotics Summative Website

This cross-platform website I've made with HTML and CSS (code repository) is made to showcase my robotics summative write-up for the 2020 First Robotics Competition.

After having worked with Hassan on our team robot's driving mechanism in C++, I decided to join Sayfullah in working on a web application called RavenScout. Thus, I began by making my first web application. Afterwards, I wrote a lengthy guide (write-up) to solidify my knowledge, and teach the material in a way I thought would be more beneficial than the the tutorial I followed. This includes my own thorough explanations, code snippets, and diagrams.

Technologies Used:

  • Flask as the web framework



  • Python for back-end



  • Dynamic HTML5 (using Jinja), a bit of JavaScript, and CSS3 all for front-end


I found this assignment challenging and very useful towards my software engineering career. It allowed me to explore different areas of software development that opened my eyes to new opportunities and technologies.
