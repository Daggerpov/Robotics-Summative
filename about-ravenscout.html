The Scouting Plugin Details

The web application we're currently trying to develop is currently named "RavenScout". It is a scouting-plugin to help out teams in the First Robotics Competition. It'll handle spreadsheet files submitted by trusted users containing scouting information for teams' robots. This data will be stored into a database
